Loading and running the program.
Compile and run using the CITS2200.jar. Run by using the alpha as an argument and place node data in stdin.

For example for MAC terminal with alpha = 0.5 and node/edge data in file.txt: 
Compile
	javac -cp $CLASSPATH:./CITS2200.jar:. *.java
Run
	java -cp $CLASSPATH:./CITS2200.jar:. Project 0.5 < file.txt
